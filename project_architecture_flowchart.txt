  3. BIG DATA PROCESSING — Databricks + PySpark + Delta Lake
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ┌──────────────────────────────────────────────────────────────────────────┐
  │                      MEDALLION ARCHITECTURE                              │
  │                                                                          │
  │  ┌─────────────────────────────────────────────────────────────────┐    │
  │  │  BRONZE LAYER  (dbfs:/delta/banking_risk/bronze/)               │    │
  │  │  • Read raw CSV from DBFS                                       │    │
  │  │  • Add ingestion_timestamp & source_file columns                │    │
  │  │  • Store as Delta table: bronze_loan_default                    │    │
  │  │  • Schema preserved (18 cols + 2 metadata)                     │    │
  │  └─────────────────────────────┬───────────────────────────────────┘    │
  │                                │                                         │
  │                     ETL / TRANSFORM                                      │
  │                                │                                         │
  │  ┌─────────────────────────────▼───────────────────────────────────┐    │
  │  │  SILVER LAYER  (dbfs:/delta/banking_risk/silver/)               │    │
  │  │  • Drop nulls (Age, Income, LoanAmount, CreditScore, etc.)     │    │
  │  │  • Remove duplicate LoanIDs                                     │    │
  │  │  • Cast columns to proper data types                            │    │
  │  │  • Standardize categorical columns (trim whitespace)            │    │
  │  │  • Business rules: Age 18-100, CreditScore 300-850             │    │
  │  │                     Income > 0, DTI 0-5                         │    │
  │  │  • Store as Delta table: silver_loan_clean                      │    │
  │  └─────────────────────────────┬───────────────────────────────────┘    │
  │                                │                                         │
  │                    ANALYTICS / RISK ENGINE                               │
  │                                │                                         │
  │  ┌─────────────────────────────▼───────────────────────────────────┐    │
  │  │  GOLD LAYER  (dbfs:/delta/banking_risk/gold/)                   │    │
  │  │                                                                  │    │
  │  │  Risk Score Formula (0–100):                                    │    │
  │  │  ┌──────────────────────────────────────────────┐               │    │
  │  │  │  (1 - CreditScore_norm) × 30  [Low CS=Risk]  │               │    │
  │  │  │+ (1 - Income_norm)      × 20  [Low Inc=Risk]  │               │    │
  │  │  │+ LoanAmount_norm        × 15  [High Loan=Risk] │              │    │
  │  │  │+ DTIRatio_norm          × 20  [High DTI=Risk]  │              │    │
  │  │  │+ InterestRate_norm      × 10  [High IR=Risk]   │              │    │
  │  │  │+ (1 - MonthsEmp_norm)   ×  5  [Short Emp=Risk] │              │    │
  │  │  └──────────────────────────────────────────────┘               │    │
  │  │                                                                  │    │
  │  │  Risk Categories:                                               │    │
  │  │    0–40  → Low Risk    (3,492 customers)                       │    │
  │  │    41–70 → Medium Risk (14,132 customers)                      │    │
  │  │    71–100→ High Risk   (2,475 customers)                       │    │
  │  │                                                                  │    │
  │  │  Gold Delta Tables:                                             │    │
  │  │  ├── gold_customer_risk_score      (scored dataset)            │    │
  │  │  ├── gold_kpi_summary              (daily KPIs)                │    │
  │  │  ├── gold_loan_default_summary     (by loan purpose)           │    │
  │  │  ├── gold_loan_purpose_risk        (purpose × risk)            │    │
  │  │  ├── gold_employment_risk_summary  (by employment type)        │    │
  │  │  ├── gold_credit_score_risk_summary (CS range × risk)         │    │
  │  │  └── gold_financial_risk_summary   (financial KPIs × risk)    │    │
  │  └─────────────────────────────────────────────────────────────────┘    │
  └─────────────────────────────────────┬────────────────────────────────────┘
                                        │
